
## 자동차 경주 게임

- [x] 공백 입력이 있으면 예외 처리
- [x] 1 ~ 5 글자가 아니면 예외 처리

- [x] 음수거나 0인 예외 처리

- [x] 차 이름, 시도 횟수를 전달 받는다.
- [x] 게임 결과와 우승자를 반환한다.

- [x] 플레이 횟수, 플레이어 별 최종 이동 거리, 우승자, 플레이한 날짜/시간을 DB에 저장한다.
